--------------------------------------RUNNING ---------------------------------------------------------------------------
To run this app you will need the following tools 
1- install node on your system 
2- install mongodb on your system 
3- install mongodb compass 
4- open the project folder in vs code
5- open terminal in vscode and run the following command
	npm i --save
this will install all the needed modules
6- to run this app run the following command
	nodemon app.js
7- open your browser and type in localhost:3000/login 
8- thats all!!!
-----------------------------------------ABOUT THE APP----------------------------------------------------------------------- 	
 This is an node app which uses following 
 1- express
 2-  mongodb/mongoose
 3- handlebars alongside with html
 4- express sessions and validations
